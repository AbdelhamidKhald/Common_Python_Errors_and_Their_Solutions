Python Quiz - Questions and Answers

Q1: What is Python?

A) A high-level programming language
B) A low-level programming language
C) An operating system
D) A type of snake
Answer: A) A high-level programming language

Q2: Which of the following is a correct way to declare a variable in Python?

A) int x = 10
B) x = 10
C) float x = 10.5
D) var x = 10
Answer: B) x = 10

Q3: Which of the following is a keyword in Python?

A) function
B) variable
C) class
D) define
Answer: C) class

Q4: Which of the following is not a valid identifier in Python?

A) my_var
B) _var
C) 9var
D) var9
Answer: C) 9var

Q5: What will be the output of the following code?

x = 5
y = "5"
print(x + y)
A) 10
B) TypeError
C) 55
D) None of the above
Answer: B) TypeError

Q6: What will be the output of the following code?

x = 10
y = 20
print(x + y * 2)
A) 60
B) 50
C) 30
D) 40
Answer: B) 50

Q7: Which operator is used for exponentiation in Python?

A) ^
B) **
C) //
D) %
Answer: B) **

Q8: What is the output of the following code?

print(10 / 3)
A) 3
B) 3.0
C) 3.33
D) 3.3333333333333335
Answer: D) 3.3333333333333335

Q9: What will be the output of the following code?

x = "Hello"
y = "World"
print(x + y)
A) Hello World
B) HelloWorld
C) Hello, World
D) None of the above
Answer: B) HelloWorld

Q10: Which of the following statements is correct?

A) 10 / 3 results in an integer.
B) 10 / 3 results in a float.
C) 10 / 3 results in a double.
D) 10 / 3 results in a complex number.
Answer: B) 10 / 3 results in a float.
